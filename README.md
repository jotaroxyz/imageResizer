## Supported formats
Works seamlessly with standard image formats (PNG, JPEG, etc.).\
(We may plan on expanding it later)

## 💾 Installation & Download
It's as easy as it sounds!\
Go to the newest [release](https://github.com/jotaroxyz/imageResizer/releases/latest), pick the right file for you operating system -> download it and that's it!\
You can work on your images straight away!
